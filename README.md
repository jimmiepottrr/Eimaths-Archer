# Eimaths Bounty Hunter Demo UX/UI

Responsive Web App prototype for the Eimaths Bounty Hunter learning game.

## Status

Local package was unpacked and tested from `eimaths_bounty_hunter_demo_uxui.zip`.

Passed checks:

- `npm run format`
- `npm run lint`
- `npm test`
- `npm run build`

## Screens included in the demo

- Landing
- Login
- Grade selection
- Quest map
- Quiz / speed math battle
- Wallet
- Rewards redemption
- Parent report

## UX/UI reference

The demo was designed using all 7 images in `ui-references` as reference direction:

1. Mobile core UX/UI concept
2. Desktop dashboard Thai
3. Mobile rewards redemption parent view
4. Mobile rewards parent view alt
5. Desktop core experience
6. Desktop rewards boss parent view
7. Mobile quest learning flow

## Run locally

```bash
npm install
npm run dev
```

Or open `index.html` when using the standalone build.
